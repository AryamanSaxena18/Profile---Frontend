## Project Title:
Profile Card UI Design

## Objective:
To design and implement a visually appealing profile card using only HTML and CSS while applying modern layout techniques.

## Description:
This project is a static user profile card interface that displays:

- Profile image
- Name and profession
- Social media icons
- Action buttons
- Engagement statistics

The layout uses Flexbox and positioning techniques to achieve proper alignment and centering.

## Skills Demonstrated:

- HTML structure building
- CSS styling and layout design
- Flexbox for alignment
- Absolute and relative positioning
- SVG integration
- UI/UX layout thinking

## Conclusion:

This project helped improve understanding of layout structuring, alignment handling, and clean UI creation using pure frontend technologies without JavaScript.
